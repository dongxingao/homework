# Trains Frontend Structure

## 目录说明

```text
trains/
├── README.md
├── data/
│   └── mock-trains.json
├── pages/
│   └── query.html
├── scripts/
│   └── query.js
└── styles/
    └── query.css
```

## 约定

- `data/`：本地 mock 车次数据
- `pages/`：车次查询页面
- `scripts/`：查询页脚本
- `styles/`：查询页样式

当前用于先把前端查询主流程跑通，后续再替换为真实接口数据。
